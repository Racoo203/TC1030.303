siguiente comando para compilar el programa:
        
> g++ main.cpp fraccion.cpp -o main.exe
> .\main.exe 

DESCRIPCIÓN DEL PROGRAMA:
    El programa consiste en crear una clase fraccion que tiene dos tipos privados: numerador
    y denominador. La clase tiene un constructor por default y un constructor con parametros,
    y los respectivos setters y getters para cada tipo de dato. Se sobrecargaron los operadores
    de suma, resta, multiplicación, división, igualdad, desigualdad, mayor que, y menor que.

    Se hicieron casos pruebas en main.cpp para probar el funcionamiento de la clase fraccion.
